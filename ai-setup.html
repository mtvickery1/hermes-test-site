Quickstart

1

Install & Bootstrap

Get Hermes running locally with the desktop app or CLI.

hermes setup
2

Configure Providers (BYOK)

Add your own keys — avoids OpenRouter prompt caching costs.

hermes config set provider=anthropic api_key=sk-... hermes config set provider=google api_key=... hermes config set provider=openrouter api_key=...
3

Enable Tools

Unlock browser, terminal, file system, and delegation.

hermes tools enable browser terminal file delegation
4

Prime Memory & Skills

Teach it your workflow once; it remembers forever.

hermes memory add "User prefers Obsidian for all notes..."
